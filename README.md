## what is this?
A side project web app to resize images.

## to use

- `npm run start` to start dev server
- `npm run build` to build IE cross-browsing available production build.

## tech stack

- React
- React-router-dom
- vanilla CSS3
- Javascript canvas
- polyfill
- Netlify CI/CD
